资讯主题：

#汽车

#交付

#增长

#新能源

#销量

10月1日，多家新能源汽车企业公布了9月份的销售成绩单。其中，哪吒汽车交付1.8万辆新车，同比增长134%，继续位列当月造车新势力交付榜榜首。今年前三季度，哪吒汽车累计销量约为11.1万辆。

理想汽车9月销量重回“三甲”位置，交付量超过1.1万辆，摆脱了8月份因车型换代带来的销量低迷（4571辆）。9月30日，理想汽车发布了全新的六座中大型SUV理想L8和五座中大型SUV理想L7。其中，理想L8将在11月初交付。

零跑汽车9月交付量约为1.1万辆，同比增长200%，是当月造车新势力中同比增速最快的车企。近期，零跑汽车动作颇多。9月28日，零跑汽车推出了中大型轿车C01。次日，零跑汽车正式在港交所上市，但上市首日即破发，当日收盘报31.90港元，跌幅为33.54%。

蔚来9月交付量约为1.08辆，与去年持平，同比微增2.4%。今年前三季度，蔚来累计交付新车3.16万辆，同比增长29.3%，创下历史新高。值得一提的是，蔚来将在10月8日公布汽车出海的新布局。

小鹏汽车环比上个月继续下滑，销量超8000辆。但今年前9个月，小鹏汽车已累计交付约9.9万辆，同比增长75%。9月，小鹏汽车推出了新车型G9，这款中大型电动SUV承载了小鹏汽车冲击更高端市场的使命。

在已公布销量的新能源车企中，广汽埃安9月销量首次突破3万辆，同比增长121%；今年前三季度，广汽埃安以18.2万辆的累计销量拿下造车新势力和汽车新品牌交付量的第一名。

问界延续了此前的高增长趋势，9月交付量约为1.01万辆，年内累计交付量约4.58万辆；极氪实现了环比的持续增长，突破8000辆，年内累计交付量约4.55万辆；岚图9月也实现了环比小幅增长，交付新车2519辆；智己汽车9月则交付了1019辆新车。

整体来看，新能源车企在9月份都积极投放新品，理想汽车L8、零跑C01、小鹏汽车G9等新车型均已亮相或上市。西南证券认为，在新能源车供给持续多元的背景下，预计新能源车渗透率有望继续保持高位。

根据乘联会数据，9月，新能源车零售销量预计为58.0万辆，同比增长73.9%，环比增长9.5%，渗透率为29.7%。乘联会方面表示，目前，新能源车市场需求强劲，头部厂商产能提升，交付周期缩短。国家补贴、地方牌照政策临近到期，促进新能源车提前购买，为总体车市提供强有力支撑。10月1日，多家新能源汽车企业公布了9月份的销售成绩单。其中，哪吒汽车交付1.8万辆新车，同比增长134%，继续位列当月造车新势力交付榜榜首。今年前三季度，哪吒汽车累计销量约为11.1万辆。
